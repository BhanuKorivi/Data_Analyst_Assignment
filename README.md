This assignment is structured to demonstrate practical skills in SQL, Spreadsheet analysis, and Python programming. The approach followed is:
Understanding the schema first to identify relationships between tables such as users, bookings, and transactions.
Writing optimized SQL queries using joins, aggregation (SUM, GROUP BY), and window functions (RANK, DENSE_RANK) to extract meaningful insights.
Applying date-based filtering to analyze data for specific months and years.
Using spreadsheet functions like XLOOKUP and COUNTIFS to perform data mapping and conditional analysis efficiently.
Implementing Python logic using loops and basic operators to solve string and time conversion problems in a simple and readable way.
Overall, the focus is on clarity, correctness, and structured problem-solving, ensuring that each solution is both efficient and easy to understand.
